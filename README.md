# 🏥 NFHS-5 India Healthcare Analysis

## Project Overview
This project analyzes India's National Family Health Survey (NFHS-5) 
2019-21 data to study healthcare disparities across Indian states.

## Tools Used
- Python (pandas, matplotlib, openpyxl)
- VS Code
- Google Sheets
- data.gov.in

## Key Findings
- Bihar female literacy only 57% — needs urgent attention
- Sanitation access varies from 49% to 99% across states
- Health insurance national average only 41%
- Kerala leads in female literacy (95%+)

## Files
- `project.py` — Data loading and cleaning
- `charts.py` — Chart generation
- `report.py` — Excel report creation
- `step4_powerpoint.py` — PowerPoint presentation

## Data Source
National Family Health Survey (NFHS-5), 2019-21
Department of Health & Family Welfare, Government of India
Source: data.gov.in
